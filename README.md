# VRP Demo

Public demonstration layer for the VRP protocol.  
Includes visualizations, mock sessions, and preview tools for understanding VRP behavior.

## Structure

- `src/websocket/` — event stream listener (planned)
- `src/mock/` — mock VRP sessions and movement flows
- `src/ui/` — minimal UI for visualizing routing behavior
- `docs/` — demo overview and usage notes

## Status

Source code is being prepared for structured release once the WebSocket event stream is finalized.
